Spotify Web Player Clone

This project is a front-end clone of Spotify's web player interface built using HTML and CSS. It replicates the layout, design, and visual elements of the official Spotify application, including navigation, playlists, trending sections, and a footer music player.

Features

Sidebar navigation with Home, Search, and Library options  
Sections for Recently Played, Trending Now, and Featured Charts  
Sticky top navigation with Premium and Install App buttons  
Footer music player with control icons  
Responsive layout using CSS media queries  
Integration of Google Fonts and Font Awesome for typography and icons  

Technologies Used

HTML5  
CSS3  
Font Awesome  
Google Fonts  

How to Run

1. Clone this repository  
   git clone https://github.com/yourusername/spotify-clone.git  

2. Navigate to the project folder  
   cd spotify-clone/CSS PROJECT  

3. Open the file spotify.html in a web browser  

Notes

This is a static front-end project and does not include music streaming functionality.  
It is intended for practicing HTML and CSS design skills and for building visually accurate UI layouts.
